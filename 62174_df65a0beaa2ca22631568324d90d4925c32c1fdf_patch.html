LU-19559 lnet: lnet_peer reduce holes reported by pahole

lnet_peer reduce holes reported by pahole. There
is no function change introduced by this patch.

$ pahole -C lnet_peer lnet/lnet/lnet.o
/* size: 336, cachelines: 6, members: 33 */
/* sum members: 317, holes: 4, sum holes: 15 */
/* padding: 4 */
/* last cacheline: 16 bytes */

After:

$ pahole -C lnet_peer lnet/lnet/lnet.o
/* size: 320, cachelines: 5, members: 33 */
/* padding: 3 */
