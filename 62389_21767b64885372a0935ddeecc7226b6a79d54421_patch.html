LU-19566 lustre: add ol_comp_flags to ost_layout

Extended struct ost_layout from 28 to 32 bytes by
adding __u32 ol_comp_flags field to communicate
component flags from MDT to OST via filter_fid.

Created versioned structures (ost_layout_217,
filter_fid_217) and conversion functions for
backward compatibility with old 28-byte layouts.

Updated LFSCK and LOD to populate ol_comp_flags
from component flags when creating or reconstructing
filter_fid, with size-based detection to handle old
and new filter_fid versions when reading from disk.

Updated wiretest checks and ll_decode_filter_fid
utility to recognize new structure sizes.
