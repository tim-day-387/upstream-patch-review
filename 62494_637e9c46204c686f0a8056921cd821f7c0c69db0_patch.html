LU-19469 utils: tool to analyze sparsity of a file

Develop a tool to analyze the sparse file hole distribution and
sparsity ratio.
It can output the total hole count, total hole bytes, sparsity
ratio, min/max hole size, and min/max hole range and hole size
histogram.
