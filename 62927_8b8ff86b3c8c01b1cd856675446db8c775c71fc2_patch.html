LU-19670 TCU: trusted.unrm xattr should be visible

Since xattr list is cached on client side, when trusted.unrm is
set on the server, the client still uses the stale cache and
causes error.  Always get trusted.unrm directly from the server.

sanityn.sh test_118f is added to verify this patch.
