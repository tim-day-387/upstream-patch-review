LU-14554 lfs: Return error for verify when stale components exist

Running "lfs mirror verify " with a stale component should print
a message such as "Mirror 2: 0x8f5a8013 (stale)" and return a non-zero
exit code. This prevents misleading results where "lfs mirror verify"
previously returned success even though the mirrors were not in sync.
