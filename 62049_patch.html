LU-12668 llapi: llapi_mirror_find() parity mirrors do not contain data

llapi_mirror_find() is used to find a non-stale mirror containing
file data for the purpose of syncing to stale mirrors.
Skip any parity mirrors as they do not hold file data to sync from.
