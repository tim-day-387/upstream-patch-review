LU-0000 tests: add SOM mirror layout tests

Add tests 70-76 to llapi_layout_test.c to verify SOM
mirror layout creation and validation:

Test 70: Verify LCME_FL_SOM cannot be set via
llapi_layout_comp_flags_set() - users must use the
dedicated llapi_layout_comp_add_som() API.

Test 71: Verify llapi_layout_comp_add_som() creates a
valid SOM component with correct properties (0-EOF
extent, SOM pattern, zero stripes, SOM flag).

Test 72: Verify stripe count and stripe size cannot be
set on SOM components.

Test 73: Verify layout sanity check detects and rejects
layouts with multiple SOM mirrors.

Test 74: Verify properly created SOM layouts pass sanity
checks.

Test 75: Verify SOM components are properly identified
by their flags.

Test 76: Verify properly created SOM layouts with
correct extents pass sanity checks.

Note: File creation tests are limited to in-memory
layout validation as server-side SOM mirror support
will be implemented in later phases.

Test-Parameters: fortestonly
Test-Parameters: forjanitoronly
