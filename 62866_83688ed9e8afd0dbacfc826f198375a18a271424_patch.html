LU-17000 lnet: Remove redundent NULL check

Remove redundent NULL check for pointer net_l.
It would be impossible for net_l to be NULL as
it would be already set after lnet_net_unique()
call to either net (network being configured) in
case it is unique (true case) or existing network
in case of not unique (false case)
