LU-0000 lfs: add --som option to lfs setstripe

Implement lfs setstripe --som to create files with a
data mirror plus a SOM (Size-on-MDS) mirror. The --som
flag creates a two-mirror file where the second mirror
is a SOM component.

The implementation follows the pattern established by
--ec (erasure coding): the SOM component is added to
the data mirror's layout, and the mirror_create()
function counts it as 2 mirrors (data + SOM).

Validation ensures:
- --som requires -N (mirror mode)
- --som cannot be used with --ec
- --som cannot be used with component operations
- --som cannot be used with foreign layouts

Update layout_pattern_supported() to recognize
LOV_PATTERN_SOM as a valid pattern so that getstripe
displays "som" instead of the hex value "1000".
