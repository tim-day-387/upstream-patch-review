LU-0000 lfs: allow --som without -N flag

Allow 'lfs setstripe --som' to work without requiring
the -N flag, similar to how --ec works. When --som is
specified without -N, automatically create a 2-mirror
file with one data mirror and one SOM mirror.

This simplifies the command line interface and makes
--som consistent with --ec behavior.

Before:
  lfs setstripe -N -E -1 -c 2 --som file

After (both work):
  lfs setstripe -N -E -1 -c 2 --som file
  lfs setstripe -E -1 -c 2 --som file
