LU-19562 llite: add FLR EC support and enable switch

Add OBD_CONNECT2_FLR_EC connect flag support and
enable_erasure_coding parameter to control EC layouts.

We add the connection flags now at the start of the
feature introduction so it can be tested, but we also
have the llite flag so the feature is not on by
default.

It will be switched to on-by-default once the feature
is fully implemented.

EC layout creation is gated on both the local
enable_erasure_coding parameter and the presence of
the OBD_CONNECT2_FLR_EC flag in the MDT connection.
Both must be set for EC layouts to be allowed.
