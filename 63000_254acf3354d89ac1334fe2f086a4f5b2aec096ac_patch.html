LU-19691 tests: skip sanity-sec/25b in local mode

Sanity-sec/25b compares nodemap output to ensure "lctl nodemap_info"
and "lctl get_param" are equivalent. If all servers are colocated on
the same node, the nodemap "exports" property list is not guaranteed
to be equivalent everytime. This makes 25b in local mode
intermittently fail.

This patch skips 25b in local mode to avoid any Janitor errors.
