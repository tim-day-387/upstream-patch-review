LU-0000 tests: add lfs mirror create tests for EC layouts

Add comprehensive test coverage for lfs mirror create with
EC (Erasure Coding) layouts. Since lfs mirror create uses
the same code path as lfs setstripe -N, these tests verify
that EC support works correctly with the mirror create
interface.

Tests added (3a-3e):
- 3a: Single EC mirror with lfs mirror create -N
- 3b: Multiple identical EC mirrors with -N2
- 3c: Mixed EC and regular mirrors
- 3d: Different EC configs per mirror
- 3e: Multiple EC specs in one mirror

All tests verify mirror count, component count, parity
flags, and EC stripe counts (dstripe/cstripe).

Test-Parameters: trivial
