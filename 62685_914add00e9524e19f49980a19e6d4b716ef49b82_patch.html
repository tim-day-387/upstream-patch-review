LU-17240 tests: unmount targets in parallel

Unmount targets (clients, OSTs, and MDTs) in parallel to
speed up test cluster teardown. This complements the
parallel formatting and mounting functionality.

The PARALLEL_UMOUNT variable controls which target types
are unmounted in parallel.  When "client" is included,
client unmounts are backgrounded and run in parallel
with server unmounts.

This change modifies stopall() and unmountoss() to unmount
targets in parallel when PARALLEL_UMOUNT is set.

Performance improvement (1 MDS, 4 OSTs, 1 client):
- Sequential unmount: 51.8 seconds
- Parallel servers only (ost mdt): 20.4 seconds (2.5x)
- Parallel clients+servers (ost mdt client): 12.0 seconds
  (4.3x)

The speedup scales with the number of targets.

Test-Parameters: testlist=conf-sanity mdscount=2 mdtcount=4
