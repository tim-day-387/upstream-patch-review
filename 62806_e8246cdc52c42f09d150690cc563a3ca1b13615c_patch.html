LU-0000 lov: skip SOM mirrors for normal read/write IOs

Add lre_som flag to track mirrors containing SOM components.
Skip these mirrors during all I/O operations, including both
normal and designated mirror IOs, since SOM mirrors contain
only metadata and have no actual data to read or write.

Unlike parity mirrors which can be read (they contain actual
data), SOM mirrors should never be used for I/O operations.
Attempting to use a SOM mirror for designated I/O returns
EINVAL.

Test 221 verifies that:
- Normal reads skip SOM mirrors and use data mirrors
- Normal writes skip SOM mirrors and use data mirrors
- Designated reads from SOM mirrors fail with EINVAL

Test-Parameters: fortestonly
