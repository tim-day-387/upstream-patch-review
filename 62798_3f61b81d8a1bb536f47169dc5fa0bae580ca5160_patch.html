LU-0000 lod: minimal LOD support for SOM mirror creation

Add minimal LOD support to allow creation of files
with SOM mirror components:

- Set LOV_PATTERN_SOM when LCME_FL_SOM flag is
  present
- Skip SOM components during stripe allocation
  (zero stripes)

This enables testing of SOM layout creation APIs.
Full LOD support (mirror selection, prefer flag
blocking) is added in a later patch.
