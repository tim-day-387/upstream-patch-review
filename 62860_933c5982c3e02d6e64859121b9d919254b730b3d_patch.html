LU-18857 mgs: add allow_register tunable parameter

Add a new tunable parameter 'allow_register' to the MGS
that controls whether new Lustre filesystem can be
registered. This provides administrators with the ability
to temporarily block new filesystem registration.

The parameter can be controlled via
 - Module parameter: /sys/module/mgs/parameters/allow_register
 - Proc interface: /proc/fs/lustre/mgs/MGS/allow_register
