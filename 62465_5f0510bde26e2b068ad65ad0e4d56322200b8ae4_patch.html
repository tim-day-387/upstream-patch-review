The lraft is imported as-is from another repo. There are
variable declarations in the middle of functions. It would
be possible to fix these errors in our copy of this code,
but it would be better to disable the
`-Werror=declaration-after-statement` warning when building
this module, so that we don't need to edit the repo every
time it is updated from upstream.
