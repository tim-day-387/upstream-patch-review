LU-19548 lfs: support mirror split -d for parity mirrors

Add support for destroying parity mirrors using 'lfs
mirror split --mirror-id N -d'. Regular split (without
-d) is blocked for parity mirrors since it doesn't make
sense to copy a parity mirror to a standalone file.
