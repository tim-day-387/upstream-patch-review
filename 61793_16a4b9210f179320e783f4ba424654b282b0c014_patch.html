LU-14554 lfs: Allow marking mirror component stale using mirror-id

Allow users to mark a specific mirror component as stale using:
    lfs setstripe --comp-set --comp-flags=stale --mirror-id=N 

Previously, users had to manually compute the component ID using:
    --comp-id=$(((mirror_id << 16) | 1))

This change simplifies the process by enabling direct use of the
mirror-id option to mark components as stale, improving usability
and reducing potential for errors.
