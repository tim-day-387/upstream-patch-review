LU-17240 tests: remove unnecessary sleep in setupall

With parallel mounting, MDTs and OSTs start simultaneously,
eliminating the need for the 5 second sleep that waited for
MDT-to-OST connections when MDTs mounted before OSTs.

The sleep is still required for GSS configurations which
need additional time for security context negotiation.

Saves 3-5 seconds in test setup time.
