LU-8066 ldlm: fix ldlm parameter validation checks

This patch addresses two issues in the parameter validation logic:

1. In lock_reclaim_threshold_mb_store(), add missing validation to
   ensure lock_reclaim_threshold_mb is less than ldlm_lock_limit_mb.

2. In lock_limit_mb_store(), remove redundant and incorrect checks
   that were preventing valid parameter updates. The previous
   comparison logic was flawed and could block legitimate
   configuration changes.
