LU-19296 changelog: Add user-specific changelog filtering

This patch implements changelog filtering based on username and
masks. The work includes:
- Define a new API llapi_changelog_start_user() to get changelog
  records based on username and masks.
- Use chlg_ioctl() with OBD_IOC_CHANGELOG_FILTER to set changelog
  filter and call new function mdc_changelog_get_user_info() to
  get changelog user information from MDT by a RPC request.
- Add options --user and --mask to "lfs changelog" command to
  support user-specific changelog reading.
- Add man page llapi_changelog_start_user.3 and update man pages
  llapi_changelog_start.3 and lfs-changelog.1 to reflect the new
  changes.
- Add sanity.sh/test_160v to verify this patch.
